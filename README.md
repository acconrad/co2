# co2: Lisp on NES/Famicom

A lisp for building NES/Famicom games.

![](shot.png)

See example.co2 for more info.

Commands:

- defvar
- defun
- defint (interrupt)
- defconst 
- loop
- do
- asm
- set!
- poke!
- peek
- +
- -
- *
- bitwise-and
- bitwise-or
- bitwise-eor
- inc
- dec
- wait-vblank
- org
- memset
- ppu-memset
- ppu-memset-carry-on
- ppu-memcpy
